A premium, cross-platform mobile application built to monitor critical environmental metrics. Designed with a sleek dark-mode UI, it provides real-time temperature telemetry, animated charts, and system status right from your mobile device.

## ✨ Key Features

- **Live Telemetry Dashboard:** View real-time temperature changes, target limits, and device battery/power status at a glance.
- **Animated Trend Charts:** Smooth, dynamic bezier line charts powered by `react-native-chart-kit` showing the latest historical data.
- **Alert Configurations:** Easily toggle notification channels including WhatsApp, SMS, and Offline fallback alerts.
- **History & Reports:** View and securely download weekly or monthly temperature logs and analytics.
- **Premium UI/UX:** Crafted with `expo-linear-gradient` and custom styling for a stunning, futuristic dark theme.

## 🚀 Tech Stack

- **Framework:** React Native, Expo
- **Navigation:** React Navigation (Bottom Tabs)
- **Data Visualization:** React Native Chart Kit
- **Icons & Styling:** Ionicons, Expo Linear Gradients

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Thineth388/Smart-Temp-Monitor.git
   cd Smart-Temp-Monitor
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Run the Application:**
   Start the Expo development server:
   ```bash
   npm start
   ```
   *Press `a` to run on Android emulator, or scan the QR code with the Expo Go app on your physical device.*
